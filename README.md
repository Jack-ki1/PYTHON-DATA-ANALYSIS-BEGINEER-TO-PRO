## Level 1A: In-depth Pandas & Numpy

This section focuses on mastering two of the most fundamental libraries for data manipulation in Python: NumPy and Pandas. It provides foundational knowledge necessary for data analysis.

### Files Included:
- **BASICS_of_PYTHON.ipynb**: Covers fundamental Python programming concepts essential for data analysis, including data types, loops, functions, and basic operations.
- **NUMPY_LIBRARY_BREAKDOWN.ipynb**: A comprehensive guide to NumPy, covering arrays, indexing, broadcasting, mathematical operations, and performance considerations.
- **PANDAS_LIBRARY_BREAKDOWN.ipynb**: An in-depth exploration of the Pandas library, including Series, DataFrame operations, data cleaning, transformation, and analysis techniques.
- **SAMPLE_IMPLIMENTATION1.ipynb**: Practical examples implementing both NumPy and Pandas for real-world data manipulation tasks.
- **S002_DATA.csv**: Sample dataset used for exercises in this section.

## Level 1B: Python Visuals Tools

This section covers data visualization techniques using popular Python libraries such as Matplotlib, Seaborn, and Plotly. Visualization is a critical skill for understanding data patterns and communicating insights effectively.

### Files Included:
- **ALL_VISUALS.ipynb**: A comprehensive notebook covering various visualization techniques using different libraries, including plots, charts, and interactive graphics.
- **PLOTLY.ipynb**: Focuses specifically on interactive visualizations using the Plotly library, enabling dynamic and web-ready graphics.
- **Seaborn_and_Matplotlib.ipynb**: Explores statistical visualizations with Seaborn and basic plotting with Matplotlib, showing how to create publication-quality figures.
- **SOO3_DATA.csv**: Dataset used for visualization examples in this section.

## Level 1C: Excel in Python

This section demonstrates how to work with Excel files using Python libraries such as pandas and openpyxl. It bridges traditional Excel workflows with Python's powerful automation capabilities.

### Files Included:
- **EXCEL_IN_PYTHON.ipynb**: Comprehensive guide to reading, writing, and manipulating Excel files using Python libraries.
- **file_for_generating_Datasets.ipynb**: Notebook focused on generating datasets in Excel format programmatically.
- Multiple Excel files (data.xlsx, file1.xlsx, file2.xlsx, formatted.xlsx, input.xlsx, multi_sheet.xlsx, output.xlsx, raw_sales.xlsx, report.xlsx, report_template.xlsx, sales_raw.xlsx): Example Excel files used for demonstrations and exercises throughout this section.

## Level 1D: Web Scraping

This section introduces web scraping techniques, teaching how to extract data from websites programmatically. It covers ethical scraping practices and handling different types of web content.

### Files Included:
- **WEBSCRAPING_COURSE.ipynb**: A comprehensive course on web scraping fundamentals, including libraries like BeautifulSoup and requests.
- **PROJECT_01.ipynb**: First project applying web scraping techniques to extract specific data from a website.
- **PROJECT_02.ipynb**: Second project exploring more advanced scraping techniques and handling complex web structures.
- **website.html**: Sample HTML file used for practicing scraping techniques.
- **scraped.csv**: Output file containing data scraped during exercises.

## Level 2A: Streamlit Introduction

This section introduces Streamlit, a powerful tool for creating interactive data applications with minimal code. It teaches how to build dashboards and data apps quickly.

### Files Included:
- **INTRODUCTION_TO_STREAMLIT.ipynb**: Comprehensive introduction to Streamlit, covering basic widgets, layouts, and app creation.
- **STREAMLIT_PROJECTS_1-10.ipynb**: Collection of 10 practical projects demonstrating various Streamlit capabilities.
- **Test_project_11.ipynb**, **Test_project_12.ipynb**, **Test_project_13.ipynb**: Additional test projects expanding on Streamlit features.
- **style.css**: CSS file for custom styling of Streamlit applications.

## Level 2B: SQL Meets Python

This section explores the integration of SQL databases with Python for advanced data analysis. It covers connecting to databases, querying, and processing SQL data with Python tools.

### Files Included:
- **IPYTHON_SQL.ipynb**: Notebooks demonstrating how to use SQL directly in Jupyter notebooks with the ipython-sql extension.
- **SQLLITE_n_PYMYSQL.ipynb**: Comparing SQLite and PyMySQL libraries for different database scenarios.
- **sql_in_python.ipynb**: General guide to working with SQL databases in Python environments.

## Libraries Used

Throughout this course, we extensively use:
- NumPy: For numerical computations
- Pandas: For data manipulation and analysis
- Matplotlib: For static, publication-quality visualizations
- Seaborn: For statistical data visualization
- Plotly: For interactive visualizations

- Requests: For HTTP requests
- OpenPyXL: For Excel file manipulation
- Streamlit: For creating data applications
- SQLAlchemy: For SQL database connections
- Scikit-learn: For machine learning implementations

## Getting Started

1. Clone or download this repository to your local machine
2. Install the required packages using pip: `pip install numpy pandas matplotlib seaborn plotly jupyter beautifulsoup4 streamlit`
3. Navigate to the directory and start with the notebooks in Level 1A
4. Progress sequentially through the levels to build your skills systematically
5. Work through each notebook, executing the cells and experimenting with the code
